Automatically assess student code submissions for correctness, style, and efficiency.
Supports multiple programming languages with instant, actionable feedback.

## How it works

1. Connect your assignment repository or upload ZIP files
2. Configure the evaluation criteria and expected outputs
3. Students receive structured feedback with line-level comments

Clone the repository and follow the setup guide in the project README.
